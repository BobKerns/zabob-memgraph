## Scattergood-Olympic Cable History

### Original Installation (1972)

- 230KV, 10-mile oil-filled cable
- Three-phase, 600-800 amps per phase
- 2000-foot segments with underground vaults
- Golden Bear High Tension Oil (100K gallons)
- Santa Monica Bay ground return

### 1989 Failure and Repair

See [scattergood-olympic-1989.md](scattergood-olympic-1989.md) for detailed narrative.

Key technical details:

- **Root cause:** TMB (Thermal Mechanical Bending)
- **Duration:** [From LADWP docs]
- **Cost:** $75M estimated (8 months downtime at $13K/hour)
- **Repair method:** Liquid nitrogen freeze plugs, 17 splice collar installations
